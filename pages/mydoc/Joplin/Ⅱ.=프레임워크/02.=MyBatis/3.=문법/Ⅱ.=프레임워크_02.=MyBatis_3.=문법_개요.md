# [MyBatis Document](https://mybatis.org/mybatis-3/ko/sqlmap-xml.html)
## Select 속성
- id = 구문을 찾기 위해 사용될 수 있는 네이스페이스내 유일한 구분자
- resultType = 리턴되는 기대타입의 패키지 경로를 포함한 전체 클래스명이나 별칭
<br>
