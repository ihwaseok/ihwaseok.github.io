## Node.js 설치 후 vue 프로젝트 생성
- vue-cli 설치 필요
- <ins>webpack</ins><sup>1</sup> 으로 생성

```cmd
cd 프로젝트 경로
npm i vue-cli -g
vue init webpack 프로젝트명
```

<br>

## 애플리케이션 실행

```cmd
npm run dev
```

<br>

---

**webpack**<sup>1</sup> = 모듈을 하나의 파일로 묶어서 네트워크 비용을 최소화 하는 모듈 번들러
