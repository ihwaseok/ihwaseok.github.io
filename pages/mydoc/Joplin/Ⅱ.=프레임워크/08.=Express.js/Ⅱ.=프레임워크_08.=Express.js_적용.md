## Node.js 설치 후 express 프로젝트 생성

```cmd
cd 프로젝트 경로
npx express-generator
```

<br>

## 종속성 설치

```cmd
npm install
```

<br>

## 애플리케이션 실행
- http://localhost:3000/

```cmd
set DEBUG=myapp:* & npm start
```
