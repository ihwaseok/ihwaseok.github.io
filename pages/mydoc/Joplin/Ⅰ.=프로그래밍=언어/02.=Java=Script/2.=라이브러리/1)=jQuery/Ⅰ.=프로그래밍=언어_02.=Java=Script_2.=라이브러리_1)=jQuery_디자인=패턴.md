## 싱글톤 패턴
- 객체의 인스턴스가 1개만 생성되는 패턴
- 메모리 낭비 방지
- HTML에 가능한 코드가 들어가지 않도록 한다

```html
<html>
	<head> ... </head>
	<body> ... </body>
<script type="text/javascript" th:inline="javascript">
$(function() {
	let _mySite = {
		init : function() {
			$('#testButton').on('click', function() {
				...
			});
		},
								
		search : function() {
				...
		}
	}
			
	_mySite.init();
	$('#testButton').trigger('click');
})
</script>
</html>
```
