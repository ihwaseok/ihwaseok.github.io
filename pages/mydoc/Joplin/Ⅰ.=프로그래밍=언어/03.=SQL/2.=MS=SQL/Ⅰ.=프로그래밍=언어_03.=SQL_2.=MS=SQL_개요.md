## Microsoft SQL Server
- 마이크로소프트의 대표적인 관계형 데이터베이스 시스템
- 윈도우 서버에서만 구동되고 `C#`과 가장 높은 호환성을 자랑하는 데이터베이스
<br>

## 특징
- 기업을 겨냥한 폐쇄소스
- ACID를 완벽하게 준수하는 엔진사용
- 트랜잭션 및 절차에 대한 높은 수준의 제어
- 클라우드 컴퓨팅 플랫폼인 `Azure` 존재
- 쿼리 실행시 자동 `COMMIT`
<br>
