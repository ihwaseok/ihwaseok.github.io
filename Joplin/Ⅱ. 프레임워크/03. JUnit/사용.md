## 사용
```java
// 테스트할 클래스
public class Calculator {
	
	public int sum(int num1, int num2) {
		return num1 + num2;
	}
}
```
```java
// Junit 클래스
public class CalculatorTest {
	
	@Test
	public void testSum() {
		Calculator calculator = new Calculator();
		assertEquals(30, calculator.sum(10, 20));
	}
}
```
