## 변수 선언
|이름|중복선언|재할당|<abbr title="선언이 어디에 있던 다른 코드보다 먼저 실행되는 것">호이스팅</abbr>|비고|
|--|:--:|:--:|:--:|--|--|
|var|O|O|O||

## 단항 연산자
|연산자|설명|
|--|--|
|delete| 객체의 속성이나 배열의 원소 제거|
|typeof| 타입을 나타내는 문자열 반환|
<br>

## 관계 연산자
|연산자|설명|
|--|--|
|in| 객체의 속성이나 배열의 인덱스, 값이 있을 경우 true 반환|
|instanceof| 지정한 객체와 타입이 일치할 경우 true 반환|
<br>
